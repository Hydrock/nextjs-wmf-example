## Remote App

Before using this app, you should run Remote app. See <https://github.com/Hydrock/wmf-remote>

## NextJS

Next JS creating command (only if host doesn't exists):

```sh
ch host
npx create-next-app@latest host --typescript --app
```

Run host:

```sh
cd host
npm run dev
```
